Introduction

The assignment given was to design a webpage about ourselves. In this assignment we lernt many things. Some of the key thing we learned is how to design a web page. We used HTML and CSS to design our web page. Web page design means to create a page we like which can be acessed from web browers. For this we use HTML. HTML(Hyper Text Mark-up Language) is code which we use to wirte in the page which we are goint to design. For designing the web page which we made by HTML we use CSS. There are different types of css which we can use to make our page attractive. The key think for the designer to keep in their mind is to make the web page eye-friendly which means our eyes doesn't get stressed out if we browse the page for a long time. In this assignment we had to make different pages such as home, about, cv, social, modelsgp and contact. All the web pages should be responsive as there are more than 65% mobile users.
